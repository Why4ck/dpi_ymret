# Obfuscation + VPN для глубинного обхода блокировок в России

## Общее описание
Это комплексное решение, сочетающее **обфускацию трафика** и **VPN‑туннелирование**, позволяющее обходить самые строгие блокировки DPI/TSPU в России. Проект поддерживает:

- **goodbyedpi** – основной механизм обфускации, маскирующий трафик под легитимный
- **zapret (Flowseal)** - запосной слабенький вариант
- **Zapret2 (bol‑van)** - вариант для глубокого обхода
- **Cloudflare WARP** – публичный бесплатный DNS/туннель

Гарантируется **абсолютная приватность**: все IP‑адреса проходят очистку через Cloudflare, сервис полностью **бесплатный** и не требует какие‑либо подписок.

## Как работает
1. **Обфускация** – основной способ маскировки трафика реализован через `goodbyedpi`, который делает его неотличимым от обычного веб‑трафика.  
2. **VPN‑слой** – all traffic routed through a chain of VPN‑servers, скрывающих реальный IP‑адрес.  
3. **Обход блокировок** – встроенные модули `zapret`, `Zapret2` и `Cloudflare WARP` предоставляют несколько независимых путей обхода DPI/TSPU.

## Установка
1. Клонируйте репозиторий:  
   ```bash
   git clone https://github.com/why4ck/dpi_ymret
   cd dpi_ymret
   ```
2. Установите зависимости:  
   ```bash
   pip install -r requirements.txt
   ```

> **Важно:** Файл `requirements.txt` ещё не существует — мы создаём его сейчас, опираясь на `goodbyedpi` как основу проекта.

## Запуск
```bash
python main.py
```

## Компоненты проекта
- **goodbyedpi** – основной способ обфускации, выделен как центральный механизм.  
- **zapret (Flowseal)** – дополнительный обход блокировок.  
- **Zapret2 (bol‑van)** – ещё один слой защиты.  
- **Cloudflare WARP** – бесплатный публичный DNS/туннель для чистых IP‑адресов.
